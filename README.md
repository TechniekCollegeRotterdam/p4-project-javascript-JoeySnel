# The Sword Fighting Game
Classic One keyboard only 2 player versus fighting game.  
Designed with a 2D pixelated style.

## Controls
**Player 1**
- Move Left = 'A'
- Move Right = 'D'
- Jump = 'W'
- Attack = 'SPACEBAR'

**Player 2**
- Move Left = 'ArrowLeft'
- Move Right = 'ArrowRight'
- Jump = 'ArrowUp'
- Attack = 'ArrowDown'
